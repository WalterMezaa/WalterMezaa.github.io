1. Introduce Windows 10’s Built-in Screenshot Tool: Snipping Tool#

Snipping Tool is one of the most useful tools available natively on Windows 10. It allows taking screenshots of a specific area of the screen, annotate it and save it as an image file. Snipping Tool can be used to take screenshots of opened windows, rectangular area, or free-form area. Besides, users can also delay a screenshot with the Snipping Tool. To use the Snipping Tool, follow these steps. First, open Snipping Tool and hit the New button. Second, choose the type of snip you want to make. Thirdly, make the snip. Fourthly, annotate and save the snip. Finally, find your snip in the Pictures > Screenshots folder. By using the Windows key + PrtScn shortcut, users can quickly take screenshots, which will be saved in the same location.

2. Breakdown of Steps Needed to Take a Screenshot on Windows 10#

  1. To take a screenshot on Windows 10 using the Windows key + PrtScn shortcut, press the Windows key + PrtScn on your keyboard. The Windows key is the one with the Windows logo between Ctrl and Alt.
  2. After pressing the shortcut, the screenshot will be saved in the Pictures > Screenshots folder.
  3. To use the Snipping Tool to take a screenshot of a specific area of the screen, use the Start menu to open the Snipping Tool.
  4. Once the Snipping Tool is open, select New and select the area of the screen you want to capture.
  5. You can then save the screenshot as an image file.
  6. To save the screenshot as a file, click the floppy disk icon, enter a name for the file, and choose the destination you want to save it to.

3. Learn How to Use the Print Screen Button#

The Print Screen button is the quickest way to take screenshots on Windows 10. It takes the screenshot of the whole display and can be used in other ways as well. To use this button for taking screenshots, here are 4 simple steps:

First, press the Windows key and the Print Screen button simultaneously. The screen will dim briefly, indicating that the screenshot was taken. It will be saved in the Pictures > Screenshots folder.

Second, if you don’t want to save screenshots in the Pictures > Screenshots folder, you can change the save location in the Settings app. Go to Settings > System > Storage > Change where new content is saved. Now, choose where you want to save the screenshots.

Third, you can also take screenshots of specific areas of the screen with the Snipping Tool. To find the Snipping Tool, type “Snipping Tool” in the search box and click it. Create a snip of any area of the screen to save or share with others.

And fourth, when the screenshot is taken, it is automatically saved as an image file. You can open the image file with the Photos app to edit it, rotate it, or crop it according to your needs.

You can also use the Print Screen button to capture your entire screen in some games. This will help you capture your gaming achievements and share them on social media. This feature is available in the Game bar, which can be activated by pressing Windows key + G.

4. Use Shortcuts to Take a Screenshot with Windows 10#

Taking screenshots of your screen should be a breeze with Windows 10. Here is a guide to capturing a screenshot on your Windows 10 device with different shortcut keys.

First, press the Windows key + PrtScn on your keyboard. This will capture the entire screen and save the screenshot in the “Screenshots” folder inside the “Pictures” folder.

Second, you can use the Snipping Tool to capture a specific area of the screen. Click the Start button or hit the Windows key to open the Start menu and type “Snipping Tool” in the search bar. Open Snipping Tool and choose the type of snip you want to create.

Third, you can also take a screenshot by pressing the Alt+PrtScn keys. This will capture the active window and copy the screenshot in the clipboard.

Fourth, if you have a Surface device, you can use the physical buttons to take a screenshot. Simply press the volume down button and the power button at the same time. The screenshot will be saved in the Pictures/Screenshots folder.
