3. Different Screenshot Techniques in Windows 10 (Using Keyboard Shortcuts, Using the Snipping Tool, Using the Game Bar)#

  1. Using Keyboard Shortcuts- Taking a screenshot on a Lenovo laptop running Windows 10 is easy. To do this, press the Fn and PrtScn buttons on the Lenovo laptop at the same time. Alternatively, press the Windows logo button and PrtScn button together. Your screenshot will be saved automatically in the Screenshots folder of your Pictures Library.
  2. Using the Snipping Tool- The Snipping tool is a Windows 10 utility that allows you to capture screenshots of windows, apps, full screen, menus, and more. To use it, open the Start menu, type Snipping Tool, and press Enter. Click the New button to capture your screenshot, and save it as an image file.
  3. Using the Game Bar- The Game Bar is a feature of Windows 10 that allows you to capture screenshots of games. To use it, press the Windows logo key + G. If prompted, select Yes to open the Game Bar. Click the camera icon to capture a screenshot of the game you are playing. The screenshot will be saved in the Videos folder of your Pictures Library.
