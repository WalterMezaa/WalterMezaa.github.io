1. Overview of What ‘Print Screen’ Is#

Print Screen menangkap konten layar dan menyimpannya sebagai gambar. Fitur ini memudahkan Anda untuk menyimpan secara instan tampilan layar Anda seperti gambar, misalnya untuk berbagi dengan teman atau sebagai catatan bagi Anda. Anda juga dapat menggunakan fitur ini untuk membuat screenshot dari aplikasi-aplikasi tertentu di Windows 8.1. Berikut adalah panduan langkah demi langkah untuk membuat screenshot pada komputer Windows 8.1:

  1. Jika Anda ingin mengambil seluruh layar, tekan tombol logo Windows + PrtScn pada keyboard Anda.
  2. Jika Anda ingin mengambil screenshot aktif saja, cukup tekan tombol PrtScn saja.
  3. Jika Anda menggunakan tablet Windows 8.1, tekan tombol logo Windows dan tombol Volume Down secara bersamaan untuk mengambil screenshot.
  4. Kedua tindakan di atas akan menyebabkan Windows menyimpan screenshot sebagai gambar di folder Gambar > Screenshots.
  5. Jika Anda ingin membuat screenshot dari aplikasi tertentu, tekan tombol ALT + PrtScn.
  6. Anda juga dapat menggunakan kombinasi tombol ALT + Fn + PrtScn untuk mengambil screenshot secara penuh layar.

2. Steps to Take a Screenshot on Windows 8.1#

  1. First, open the window you want to take a screenshot of.
  2. Press Windows logo key + PrtScn or PrtScn button on your keyboard to capture the screen.
  3. If you are on a tablet, press the Windows logo button and the Volume down button simultaneously to capture the screen.
  4. To view your screenshot, open the Pictures library folder and look for a Screenshots folder. Your screenshots will automatically get saved in this folder.
  5. To save the screenshots that you capture in any other folder, go to Settings, click on the Ease of Access option, and then select Keyboard.
  6. Now enable the Use the PrtScn Button to open the Snipping Tool option. This will save your screenshots in the folder of your choice.

3. Overview of the Screenshot Captured#

  1. To capture the entire window, press the ‘Windows Logo Key + PrtScn’ or ‘PrtScn’ button on keyboard. The image will be captured and automatically stored in the ‘Screenshots’ folder located in the ‘Pictures’ folder.
  2. To capture the active window only, press the ‘Alt + PrtScn’ keys simultaneously. This will capture the current active window and store the image in the ‘Screenshot’ folder.
  3. To capture the entire screen, press the ‘Windows Logo Key + Shift + S’ key. This will bring up the screenshot capturing box. Moving the cursor allows you to select the area you want to capture and press ‘Enter’. The captured image will be stored in the clipboard.
  4. To print the captured screen, open the image using a photo-editing software and select the Print option.
  5. For tablets, press the ‘Windows Logo Button’ and ‘Volume Down’ button simultaneously to take a screenshot and it will be automatically saved in the ‘Pictures’ folder in the ‘Screenshots’ folder.
  6. To view the captured screenshots, open the ‘Pictures’ library and look for the ‘Screenshots’ folder. This folder contains all screenshots captured by the user since first use of the device.
