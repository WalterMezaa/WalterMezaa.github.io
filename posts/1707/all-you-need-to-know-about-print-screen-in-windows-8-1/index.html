To change the location where the screenshots are saved, you can access the ‘Screenshots’ folder by going to the ‘Pictures’ folder in Windows 8.1. Then, right-click on the ‘Screenshots’ folder and select ‘Properties’. On the ‘Location’ tab, you can then change the ‘Target’ location, to save the screenshots to your desired folder.
